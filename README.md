# logo

seismo-learn 的 logo 由三部分组成：

- 震源球
- 地震波形
- 汉字“学”

## 预览与下载

- [PDF 格式](logo.pdf)
- [PNG 格式](logo.png)
- [JPG 格式](logo.jpg)

## 相关脚本

- [`make_waveform.sh`](make_waveform.sh): 绘制 IU.HRV 台记录的南美大地震的波形

此 logo 目前已用于如下场景：

- [GitHub 组织页面](https://github.com/organizations/seismo-learn/settings/profile)
